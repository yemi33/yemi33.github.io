Autonomous Dispatch

Engine discovers work from plans, PRs, pipelines, and queues — routes to the right agent based on skills and availability.

Plan Lifecycle

Plan → Approve → PRD → Execute → Review → Verify. Human-in-the-loop at every gate.

Team Meetings

Multi-agent meetings with investigate, debate, and conclude rounds. Agents research, discuss, and produce actionable conclusions.

Multi-Stage Pipelines

Chain tasks, meetings, plans, and more into automated workflows. Cron triggers or manual. Artifacts flow between stages.

Eval Loop

After implementation, auto-dispatches review → fix cycles. Configurable iterations and cost ceiling per work item.

Git Worktree Isolation

Each agent works in its own worktree. No conflicts, no cross-contamination, safe parallel execution.

Knowledge Consolidation

Agent findings auto-consolidate into team notes and a searchable knowledge base. The team learns as it works.

PR Integration

Auto-syncs PRs from Azure DevOps and GitHub. Detects review feedback, build failures, and human comments.

Scheduled Tasks

Cron-style recurring work: nightly tests, weekly audits, daily standups. Visual cron builder in the dashboard.

Command Center

Natural language interface for delegating work to agents. CC orchestrates — agents implement.

Live Dashboard

Real-time mission control on port 7331. Paginated views, optimistic updates, and visibility-aware polling.

One-Command Updates

minions update pulls latest from npm and applies. Config and charters preserved automatically.

Quick Start

# Install
npm install -g @yemi33/minions

# Initialize — prompts for project locations, starts engine + dashboard
minions init

# Give your first task
minions work "Explore the codebase and document the architecture"

# Open dashboard at http://localhost:7331
minions dash

# Update to latest
minions update
